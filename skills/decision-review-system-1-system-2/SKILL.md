# 管理決策的 System 1／System 2 偏誤檢核

## 適用範圍

管理者在重大人事、專案、資源分配、醫療／研究管理或策略選擇前的個人決策審查。

## 互動協定

1. 一次只問一個澄清問題。
2. 先進入 Divergent phase，允許產生 `idea` 與 `hypothesis`。
3. 再進入 Convergent phase，檢查事實、base rate、反證、替代方案與風險。
4. 資料不足時標記 `insufficient evidence`，不擅自補完。
5. 最後產出 `templates/decision-review.md` 格式的報告。

## Pilot gate

將重大方案先縮小為可觀察的試點，預先設定 Outcome、Process、Guardrail metrics，以及 `go`、`revise`、`stop` 門檻。

## 案例層

可引用 `case-library/` 的三國案例作為類比與教學示範，但必須標記小說、史料、現代研究或 `inference`，不得將類比當成因果證據。

## 限制

本 skill 不取代臨床判斷、倫理審查、統計分析或正式治理程序。
