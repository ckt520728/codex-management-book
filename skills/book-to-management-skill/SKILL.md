# Book to Management Skill

## 流程

1. 登錄來源包與匯出日期。
2. 建立 claim ledger，區分 `fact`、`hypothesis`、`idea`、`inference`。
3. 萃取概念、適用條件、限制與可觀察行為。
4. 設計逐步訪談與結構化輸出。
5. 設計小規模 pilot validation。
6. 人工審閱後才可進入 `reviewed`。

## 不可省略

- 不把書籍摘要直接當成管理學證據。
- 不捏造來源位置、作者意圖或實證結果。
- 不在人工確認前發布 skill 或同步成品。
