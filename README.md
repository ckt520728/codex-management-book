# Codex Management Book

以管理學經典書籍建立可重複使用的 Codex skills，並將決策流程轉化為可驗證的互動式工具。

## 目前範本

- Daniel Kahneman, *Thinking, Fast and Slow*
- 第一個 skill：管理決策的 System 1／System 2 偏誤檢核
- 目前狀態：`draft`

## 目錄

- `books/`：來源包與登錄資料
- `notes/`：證據與管理概念筆記
- `skills/`：可重複使用的 skills
- `apps/`：互動式網頁
- `integrations/`：四個外部系統的連線說明
- `templates/`：可重用模板
- `validation/`：試點與人工審核紀錄
